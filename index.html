Write to tables

Write to database tables by defining the model at runtime
// Create a new customer
customer = TableOperation.CreateNewRow(     "Adventureworks", "Customer", customer);
// Update an existing customer
customer = TableOperation.UpdateRow(     "Adventureworks", "Customer", customer);
Or use the RESTful Web API
POST /api/AdventureWorks/_table/Customer

--header 'Content-Type: application/json' -d
{
  "Title": "Mr.",
  "FirstName": "Jeremy",
  "LastName": "Los"
}
PUT /api/AdventureWorks/_table/Customer/42

--header 'Content-Type: application/json' -d
{
  "FirstName": "James"
  "LastName": "Lot"
}

Read from tables

Read from tables by defining the model at runtime
// Retieve a list of customers
var customers = TableOperation.Get(     "Adventureworks", "Customers");
// Retieve a single customer
var customer = TableOperation.GetById(     "Adventureworks", "Customers", "42");
Or use the RESTful Web API
GET /api/AdventureWorks/_table/Customers/42
GET /api/AdventureWorks/_table/Customers

Delete from tables

Delete a row from a table
// Delete a customer
TableOperation.DeleteRow(     "Adventureworks", "Customer", "42");
Or use the RESTful Web API
DELETE /api/AdventureWorks/_table/Customer/42

Add a table to a database

Create a new table in the database
// Create a Table
DatabaseOperation.CreateTable(     "Adventureworks", "Customer", "42");
Or use the RESTful Web API
DELETE /api/AdventureWorks/_table/Customer/42

Add a column to a table

Create a new table in the database
// Create a column

Auto Generated API's (No Code Required)

Instant. Queryable. Multi-platform.

Instant Generation

Enter the connection string to your backend data source and the EDC's dynamic data access service instantly generates APIs for your data and database objects (e.g. Tables, Views, Stored Procedures, Schemas, etc.). Changes to your backend schema are immediately reflected in the API.
See how it works.

Fully Queryable

All our REST endpoints are OData compliant. OData is an open protocol to allow the creation and consumption of queryable and interoperable RESTful APIs in a simple and standard way.

Any Database

The EDC supports multiple popular SQL databases including MS SQL, MySQL, and Oracle.
