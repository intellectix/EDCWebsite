EDC

A simple dynamic data access service for your application. Change your database schemas on the fly, without having to recompile. Supports multiple databases. .NET native or run as a service with a full set of APIs.

Instant. Queryable. Multi-platform.

Instant Generation

Enter the connection string to your backend data source and the EDC's dynamic data access service instantly generates APIs for your data and database objects (e.g. Tables, Views, Stored Procedures, Schemas, etc.). Changes to your backend schema are immediately reflected in the API.
See how it works.

Fully Queryable

All our REST endpoints are OData compliant. OData is an open protocol to allow the creation and consumption of queryable and interoperable RESTful APIs in a simple and standard way.

Any Database

The EDC supports multiple popular SQL databases including MS SQL, MySQL, and Oracle.
