Features [Pro]

In addition to the free features availabile in EDC Core, the Pro version of the EDC contains the following enhancements:

EDC Audit

The Logbook is a configurable service that captures all the audit and request history data for the EDC. The Logbook is accessible via the Hub, putting insight to your data right at your fingertips.

Read our Case Study

EDC Cache

The EDC Cache boosts the performance of the EDC by caching your databases’ schema.

EDC Identity

EDC Identity allows your applications to authenticate users against all of today’s most used identity stores (Azure AD, Active Directory, SQL, mySQL, Oracle, AD LDS, Google, Facebook, etc.).

EDC Role Manager Authentication

The EDC Role Manager allows you to permission your Web APIs, or any data source connected to the EDC Core via an easy to use role based access control (RBAC) system. Role Manager is accessible via the Hub, and comes with EDC Identity.

Report Server [Coming Soon]

Report Server provides a Data Warehouse for the EDC Insights / Logbook.
